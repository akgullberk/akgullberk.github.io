UniVento: Üniversite Etkinlik ve Kulüp Yönetim Platformu

React · JavaScript · Firebase Authentication · RESTful API · React Router · CSS · Component-Based Architecture

GitHub ⚙️Backend
Kısa Özet
UniVento, üniversite kulüpleri ve etkinlik yönetimini dijitalleştiren bir web platformu. Öğrenciler ve kulüp başkanları için kullanıcı dostu bir sistem geliştirdim. Firebase Authentication ile güvenli kimlik doğrulama, RESTful API entegrasyonu ile backend iletişimi ve component-based yapı ile modüler kod organizasyonu uyguladım. Proje, farklı kullanıcı rolleri (öğrenci/başkan) için özelleştirilmiş yetkilendirme ve iş akışları içeriyor.
Projenin Amacı
Temel hedef, üniversite kampüslerindeki kulüp ve etkinlik yönetim süreçlerini dijitalleştirmekti. Öğrenciler tüm etkinlikleri tek platformdan görüntüleyebilmeli, kulüplere üye olabilmeli ve etkinliklere katılabilmeliydi. Kulüp başkanları ise etkinlik oluşturma, üyelik başvurularını yönetme ve katılım isteklerini onaylama gibi yönetim işlemlerini gerçekleştirebilmeliydi.

Projede, React'in component-based mimarisini kullanarak her sayfayı bağımsız bir modül haline getirdim. Firebase Authentication ile kullanıcı kimlik doğrulamasını backend'den bağımsız yönetirken, RESTful API ile dinamik veri akışı sağladım. LocalStorage kullanarak hafif state yönetimi uyguladım; bu sayede küçük ve orta ölçekli bir uygulama için gereksiz state management kütüphanelerinden kaçınarak performansı korudum.
Mimari & Yapı
Uygulama, component-based mimari ile organize edildi ve üç ana katmana ayrıldı:

Presentation (Sunum) Katmanı: React component'leri ve sayfa yapıları Her sayfa (pages/) kendi state yönetimi ve lifecycle hook'ları ile bağımsız çalışır Ortak component'ler (components/) farklı sayfalarda yeniden kullanılır React Router DOM ile client-side routing sağlanır

Authentication (Kimlik Doğrulama) Katmanı: Firebase Authentication entegrasyonu Firebase Auth ile kullanıcı kaydı, giriş ve oturum yönetimi Role-based access control (kulüp başkanı/öğrenci ayrımı) Auth state değişikliklerini dinleyen reactive yapı

Data (Veri) Katmanı: RESTful API entegrasyonu Fetch API ile backend iletişimi CRUD işlemleri (etkinlik oluşturma, listeleme, silme, katılım) FormData ile çoklu form verisi ve dosya yükleme desteği
