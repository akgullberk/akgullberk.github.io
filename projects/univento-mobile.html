UniVento: Üniversite Kulüp Etkinlik Yönetim Mobil Uygulaması

React Native · JavaScript · TypeScript · Firebase Authentication · Firebase Firestore · React Navigation · Context API · Axios · AsyncStorage

GitHub
Kısa Özet
UniVento, üniversite kulüplerinin etkinliklerini yönetmek ve öğrencilerin bu etkinliklere katılımını kolaylaştırmak için geliştirdiğim cross-platform mobil uygulamadır. React Native ile iOS ve Android için tek kod tabanından geliştirildi. Firebase Authentication ile güvenli kimlik doğrulama, Context API ile durum yönetimi ve RESTful API entegrasyonu ile backend iletişimi sağlandı. Rol tabanlı erişim kontrolü (kullanıcı/kulüp başkanı) ile farklı kullanıcı rollerine özel özellikler sunuldu.
Projenin Amacı
Üniversite kulüplerinin etkinlik yönetimini dijitalleştirmek ve öğrencilerin etkinliklere katılımını kolaylaştırmak. Kulüp başkanları etkinlik oluşturabilir, katılım ve üyelik başvurularını yönetebilir. Öğrenciler etkinlikleri keşfedebilir, detaylarını görüntüleyebilir ve katılım isteği gönderebilir. Context API ile global durum yönetimi, AsyncStorage ile yerel veri saklama ve React Navigation ile akıcı navigasyon sağlandı. Modüler yapı ile bakım ve genişletme kolaylaştırıldı.
Mimari & Yapı
Uygulama, katmanlı mimari prensiplerine uygun olarak geliştirildi:

Context Layer (Durum Yönetimi): AuthContext ile global kimlik doğrulama durumu ve kullanıcı rolü yönetimi. Firebase Authentication entegrasyonu ve AsyncStorage ile kalıcı oturum yönetimi.

Service Layer (API Katmanı): Axios tabanlı servis modülü ile RESTful API çağrıları. Etkinlik, kulüp, katılım ve üyelik işlemleri için merkezi API yönetimi.

Presentation Layer (UI Katmanı): Screen-based navigasyon yapısı ile modüler ekran organizasyonu. React Navigation ile stack ve bottom tab navigasyon. Her ekran kendi stil ve mantık dosyalarına sahip.

Component Layer (Bileşen Katmanı): Yeniden kullanılabilir bileşenler (BottomBar) ve özel ekran bileşenleri. React Native Vector Icons ile tutarlı ikon kullanımı.

Data Persistence: AsyncStorage ile kullanıcı rolü ve oturum bilgilerinin yerel saklanması. Firebase Firestore ile cloud veri yönetimi.
