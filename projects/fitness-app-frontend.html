Fitness App Frontend: React ile Modern Fitness Takip Uygulaması

React · Redux Toolkit · Material-UI · Keycloak OAuth2 PKCE · React Router · Axios · Vite

Kısa Özet
Bu projede, fitness aktivitelerini görselleştiren ve yöneten, modern React teknolojileriyle geliştirilmiş bir frontend uygulaması geliştirdim. Sadece basit bir liste görünümüyle sınırlı kalmayıp, Keycloak ile güvenli kimlik doğrulama, Redux ile merkezi state yönetimi ve Material-UI ile modern bir kullanıcı arayüzü sunan kapsamlı bir yapı kurdum. Uygulama, aktivite ekleme, listeleme, detay görüntüleme ve AI destekli önerileri görüntüleme gibi işlevleri kullanıcı dostu bir şekilde yerine getiriyor.
Projenin Amacı
Temel hedef, mikroservis mimarisine uygun, güvenli ve kullanıcı dostu bir frontend uygulaması geliştirmekti.

Proje kapsamında, Keycloak ile OAuth2 PKCE akışını entegre ederek güvenli kimlik doğrulama sağladım. Kullanıcılar Keycloak üzerinden giriş yapıyor ve JWT token'ları otomatik olarak yönetiliyor.

Redux Toolkit ile merkezi state yönetimi kurarak, kullanıcı kimlik bilgileri ve token'ları uygulama genelinde tutarlı bir şekilde yönetiyorum. LocalStorage ile token'lar kalıcı hale getiriliyor ve sayfa yenilemelerinde oturum korunuyor.

Material-UI ile modern ve responsive bir kullanıcı arayüzü tasarladım. Aktivite formları, liste görünümleri ve detay sayfaları tutarlı bir tasarım diliyle sunuluyor.

React Router ile client-side routing yaparak, kullanıcıların sayfa yenilemeden farklı sayfalar arasında gezinmesini sağladım. Protected route yapısı ile yetkisiz erişimler engelleniyor.

Axios interceptors ile her API isteğine otomatik olarak JWT token ve kullanıcı ID'si ekleniyor. Bu sayede her istekte manuel token yönetimi yapılmasına gerek kalmıyor.
Mimari & Yapı
Uygulama, modern React best practice'lerine uygun olarak geliştirildi:

Authentication (OAuth2 PKCE): Keycloak ile güvenli kimlik doğrulama. react-oauth2-code-pkce kütüphanesi ile PKCE akışı implemente edildi. Kullanıcı giriş yaptığında JWT token alınıyor ve Redux store'a kaydediliyor.

State Management (Redux Toolkit): Merkezi state yönetimi. Auth slice ile kullanıcı bilgileri, token ve userId yönetiliyor. LocalStorage ile state kalıcı hale getiriliyor.

Routing (React Router): Client-side routing. Aktivite listesi, aktivite detayı ve ana sayfa route'ları tanımlandı. Protected route yapısı ile yetkisiz kullanıcılar login sayfasına yönlendiriliyor.

API Integration (Axios): Backend servisleriyle iletişim. Axios instance'ı ile base URL ve interceptor'lar yapılandırıldı. Her istekte Authorization header'ına JWT token ve X-User-ID header'ına kullanıcı ID'si otomatik ekleniyor.

UI Components (Material-UI): Modern ve tutarlı kullanıcı arayüzü. ActivityForm, ActivityList ve ActivityDetail component'leri Material-UI bileşenleriyle oluşturuldu. Responsive tasarım ile farklı ekran boyutlarına uyum sağlanıyor.

Component Structure: Modüler component yapısı. Her component kendi sorumluluğuna odaklanıyor. ActivityForm aktivite ekleme, ActivityList aktivite listeleme, ActivityDetail aktivite detayı ve AI önerilerini görüntüleme işlevlerini yerine getiriyor.

Error Handling: API hatalarının yakalanması ve kullanıcıya anlamlı mesajlar gösterilmesi. Try-catch blokları ile hata yönetimi yapılıyor.

Build & Development (Vite): Modern build tool. Vite ile hızlı development server ve optimize edilmiş production build. React 19 ve modern ES6+ özellikleri destekleniyor.
