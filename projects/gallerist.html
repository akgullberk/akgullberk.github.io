Java · Spring Boot · Spring Security · Spring Data JPA · PostgreSQL · JWT · Maven · Lombok · TCMB API

GitHub Medium
Kısa Özet
Bu projede, bir otomobil galerisinin tüm operasyonel süreçlerini (araç stoğu, müşteri yönetimi, satış takibi) dijitalleştiren, güvenliği ve performansıyla öne çıkan kapsamlı bir backend sistemi geliştirdim.

Sadece veri yönetimiyle sınırlı kalmayıp, Merkez Bankası (TCMB) API entegrasyonu ile döviz kurlarını anlık takip eden ve JWT tabanlı gelişmiş bir kimlik doğrulama mekanizmasına sahip olan modüler bir yapı kurdum.

Sistem, araç ekleme/güncelleme, müşteri kayıtları, satış işlemleri ve finansal raporlama gibi kritik işlevleri güvenli ve performanslı bir şekilde yerine getiriyor.
Projenin Amacı
Bu çalışmayı gerçekleştirirken temel hedefim, kurumsal ölçekte bir uygulamanın ihtiyaç duyduğu tüm katmanları (güvenlik, veri tutarlılığı, hata yönetimi ve dış servis entegrasyonu) profesyonel standartlarda bir araya getirmekti.

Proje kapsamında, karmaşık veritabanı ilişkilerini (One-to-Many, Many-to-Many) Spring Data JPA ile optimize ederek yönettim. Özellikle araç satış süreçlerinde finansal doğruluğu sağlamak adına dış servis entegrasyonu yaparak projenin gerçek hayat senaryolarına uygunluğunu artırdım.

Kullanıcıların güvenliğini en üst seviyede tutmak için stateless bir yapıda çalışan JWT ve Refresh Token mekanizmalarını implemente ettim. Bu sayede her istek bağımsız olarak doğrulanıyor ve oturum yönetimi sunucu tarafında herhangi bir state gerektirmiyor.

TCMB API entegrasyonu ile araç satış işlemlerinde güncel döviz kurlarını kullanarak, TL, USD ve EUR gibi farklı para birimlerinde işlem yapabilme yeteneği kazandırdım.

Proje boyunca clean code prensiplerine, SOLID prensiplerine ve katmanlı mimari yaklaşımına sadık kalarak, sürdürülebilir ve test edilebilir bir kod tabanı oluşturmaya özen gösterdim.
Mimari & Yapı
Uygulama, sürdürülebilirliği ve test edilebilirliği artırmak adına katmanlı mimari (Layered Architecture) ve DTO (Data Transfer Object) tasarım kalıpları üzerine inşa edildi:

Presentation (Controller): RESTful API standartlarına uygun, RootEntity yapısıyla standardize edilmiş cevaplar dönen uç noktalar. Tüm HTTP istekleri bu katmanda karşılanıyor ve uygun service metodlarına yönlendiriliyor. Controller sınıfları minimum business logic içeriyor ve sadece request/response yönetimine odaklanıyor.

Service: Tüm iş mantığının (Business Logic) toplandığı, dış API çağrılarının ve validasyonların yönetildiği motor katman. Araç ekleme, müşteri kaydı, satış işlemi gibi tüm operasyonlar burada koordine ediliyor. TCMB API ile döviz kuru sorgulama, JWT token üretme ve validasyon işlemleri de bu katmanda gerçekleştiriliyor.

Data Access (Repository): Spring Data JPA ile veritabanı işlemlerinin soyutlandığı ve PostgreSQL ile haberleşen katman. Custom query'ler, native SQL sorguları ve JPA repository metodları burada tanımlanıyor. Entity ilişkileri (Car-Customer, Sale-Car gibi) bu katmanda yönetiliyor.

Security: Spring Security ve JWT ile korunan uç noktalar, yetkilendirme filtreleri ve güvenli kimlik doğrulama akışı. Her istek için JWT token kontrolü yapılıyor, geçersiz veya süresi dolmuş tokenlar reddediliyor. Refresh token mekanizması ile kullanıcılar tekrar giriş yapmadan yeni access token alabiliyorlar.

Exception Handling: Uygulama genelinde oluşabilecek tüm hataların merkezi bir noktadan yakalanıp anlamlı mesajlara dönüştürüldüğü Global Exception Handler yapısı. Business exception'lar, validation hataları ve sistem hataları ayrı ayrı ele alınıyor ve kullanıcıya uygun HTTP status kodlarıyla birlikte dönülüyor.

External Integration: TCMB API ile döviz kuru entegrasyonu, RestTemplate kullanılarak gerçekleştirildi. API'den gelen XML verileri parse ediliyor ve veritabanında cache'leniyor. Bu sayede her satış işleminde dış servise gitmeden güncel kurları kullanabiliyoruz.

Data Model: PostgreSQL üzerinde normalize edilmiş tablo yapısı. Car, Customer, Sale, User gibi entity'ler arası ilişkiler JPA annotations ile tanımlanmış. Cascade işlemler ve orphan removal stratejileri doğru bir şekilde yapılandırılmış.
