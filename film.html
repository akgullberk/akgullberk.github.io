Film Uygulaması

Kullanıcıların film/dizi keşfetmesini, aramasını ve favori listeleri tutmasını sağlayan Flutter tabanlı uygulama.

Flutter Firebase Auth Firestore TMDB API Offline Cache

Kısa Özet

TMDB API ile içerik çeker, arama/filtreleme yapar, favorileri bulutta senkronize eder; çevrimdışı mod için yerel önbellek kullanır.

Projenin Amacı

Hızlı, akıcı ve offline-first bir medya keşif deneyimi sunmak; kullanıcı oturumu ve favorileri güvenli şekilde saklamak; görsel ağırlıklı veriyi performanslı yönetmek.

Kullanılan Teknolojiler

Flutter Dio Riverpod Firebase Auth Firestore SQFlite TMDB API

Mimari & Yapı

  • Clean Architecture + Repository pattern
  • Dio tabanlı servis katmanı, hata/boş durum yönetimi
  • Riverpod ile state yönetimi ve caching stratejileri
  • SQFlite ile offline okuma; Firestore ile senkron favoriler
